Break the Prison

Description: You are a high profile criminal in a secret vault. A guard protects the prison using rigid rules and simple checks. Outsmart him and escape the prison.
Topics: SQL Injection + Cryptography
Hint 1: Missing parameter: uid
Hint 2: There can be multiple prisoners.

Solution: Break the Prison simulates an internal police system used to access prisoner records and a secure vault. The application exposes two endpoints:
/profile?uid= — fetches prisoner details
/vault?secret= — decrypts the flag if the correct clearance is provided

BLOCKED_KEYWORDS = [

Pure numeric input is also blocked.
